# Privacy

墨声当前版本只做语音输入，不做聊天助手，也不默认保存转写历史。

## API Key

- API Key 只保存在当前 Windows 用户账户下。
- 保存方式使用 Windows DPAPI。
- 设置界面只显示保存状态和末尾几位。
- 项目不会把 API Key 写入仓库或上传到墨声服务器。

## Audio

- 录音只在本地内存中暂存。
- 转写时会生成临时 wav 分段文件。
- 每段请求结束后会删除临时文件。
- 当前转写服务为智谱 `GLM-ASR-2512`，音频会发送到智谱 API。

## Clipboard

墨声通过临时写入剪贴板并发送粘贴快捷键来把识别结果输入到当前光标。默认会尝试恢复之前的文本剪贴板内容。
